Monitoramento do Sistema

Veja informações em tempo real sobre o uso de recursos do seu servidor Flask.

Uso do CPU

{{ cpu_percent }}%

Indica a porcentagem de uso do processador neste momento. Valores altos podem indicar que o sistema está processando muitas tarefas simultaneamente.
Uso da Memória RAM

{{ ram_percent }}%

Mostra quanto da memória RAM está sendo utilizada. Se estiver próximo de 100%, o sistema pode ficar lento.
Outros Dados
  • Tempo ligado: {{ uptime }}
  • Processos ativos: {{ processes }}
  • Uso de disco: {{ disk_percent }}%
Tempo ligado: quanto tempo o sistema está em funcionamento.
Processos ativos: quantidade de programas/processos rodando.
Uso de disco: porcentagem de espaço utilizado no disco principal.
